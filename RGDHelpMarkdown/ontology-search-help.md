# RGD Ontology Search Help

## Overview

An ontology is a controlled vocabulary of well-defined terms with specified relationships between them capable of interpretation by both humans and computers. RGD provides comprehensive ontology search capabilities across multiple ontologies.

## Accessing Ontology Search

### Method 1: Direct Access
Access the ontology search directly from the RGD home page.

### Method 2: Via Data Landing Page
1. Mouse over "Data" in the top black banner
2. Click "Ontologies" in the pull-down menu
3. Or click "Data" to access the RGD Data page, then select "Ontologies"

## Search Features

### Search Parameters
- Accepts single words or portions of phrases
- Searches across all available ontologies
- Searches terms, synonyms, and accession numbers (database IDs)

### Example Search
Using "angiogenesis" returns a list of ontologies where the term appears with matching term counts.

## Search Results Navigation

### Results Display
The left-hand side shows ontologies with matching counts. Clicking an ontology name displays:
- Results table with associated terms
- Accession numbers for each term
- Annotation counts for terms and their children
- Links to ontology browser and ontology report page

### Icons and Navigation
- **Branch icon (green)** - Links to ontology browser page showing parent, sibling, child terms, synonyms, and graphical term relationship structure
- **Annotation icon** - Links to ontology annotations report page displaying all objects associated with the term

### Results Management

**Pagination:**
- Type page number in "Go to Page" box and click "Go"
- Use arrow icons under "Page # of #"

**Results Per Page:**
- Modify number of results using "View Results" dropdown menu

**Return to All Results:**
- Click "View All Results" button on left-hand side

## Ontologies Currently Searchable

- Gene Ontology (GO) - Biological process, cellular component, molecular function
- Disease Ontology and RGD Disease Ontology
- Pathway Ontology
- Clinical Measurement Ontology
- Measurement Method Ontology
- Experimental Condition Ontology
- Rat Strain Ontology
- Chemical Entities of Biological Interest (ChEBI)
- Mammalian Phenotype Ontology
- Human Phenotype Ontology
- Vertebrate Trait Ontology
