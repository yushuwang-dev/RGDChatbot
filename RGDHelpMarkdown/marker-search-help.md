# RGD Marker Search Help

## Overview

The Rat Genome Database provides tools to locate marker information through multiple search methods.

## Primary Search Methods

### General/Keyword Search

Available at the top of every RGD page, users can search by:
- Marker symbol
- Marker alias
- Chromosomal position
- UniSTS ID
- Other identifiers

Results display clickable numbers for different categories, including marker SSLP reports and strain information.

### Marker Search Tool

Accessed via Data menu > Marker Search.

**Search Parameters:**
- **Keyword box** - Accepts specific marker symbols or wildcard searches using asterisks
- **Gene symbol entry** - Returns associated SSLP records
- **Chromosome specification** - Filters by chromosome location
- **Position entry** - Allows base pair start and stop ranges
- **Species selection** - Rat, mouse, or human data

## Search Results Features

### Selection and Analysis
- Checkboxes beside marker symbols enable batch selection
- Genome Viewer (GViewer) tool displays relative positions of selected markers on chromosome charts

### Data Export
- Excel download function available for result sets

### Navigation
Clicking marker symbols opens detailed report pages containing comprehensive SSLP information.
